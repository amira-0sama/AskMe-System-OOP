# AskMe System (OOP Refactoring)

This is a structured C++ console application for the AskMe System, refactored from a single file into a clean Object-Oriented Programming (OOP) architecture.

 Project Structure
- Helper: File handling and input utilities.
- User & UsersManager: Managing user accounts, authentication, and signup.
- Question & QuestionsManager: Handling questions, thread replies, and system feed.
- AskMeSystem: The main controller orchestrating the system flow.
